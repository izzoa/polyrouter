Implement the approved dashboard design ("Polyrouter Prototype", Claude Design) as a fully interactive SolidJS SPA running on simulated data: Overview with live request feed and charts, Requests with filters and the routing-decision inspector (decision trace, structural features, price snapshots), Costs, Agents (key mint/rotate + harness snippets), Providers, Routing (drag-to-reorder tier chains, auto-layer toggles, header rules), Limits, Settings (notification channels with test-send), a three-step setup guide, and light/dark themes. All data flows through one simulator module that later backend changes replace with real APIs.
