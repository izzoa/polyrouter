## 3. Proxy orchestration

- [x] 3.1 `ProxyService`: factor the `decision.chain → {attempts, meta}` builder into a helper and **add `tierKey` to `AttemptMeta`** (each member's tier). `prepare`: after `structural.evaluate(...)` — `route` → decision (#13); `ambiguous` + `cascade.enabled` + `cascade.plan(snapshot)` → build cheap + strong bundles; the **escalation** chain = `strong.attempts ++ primary.attempts` (primary = the Layer-0 default bundle already built) with `escalationMeta = strong.meta ++ primary.meta` (each meta tagged with its tierKey, so a served default member records `tier_assigned=default`). `Prepared` gains optional `cascade: { cheap, escalation, cheapTimeoutMs }`; else default stands.
- [x] 3.2 `completion` (non-streaming, cascade): run `runBufferedChain(cheap, cheapSignal)` where `cheapSignal = AbortSignal.any([signal, AbortSignal.timeout(cheapTimeoutMs)])`. If cheap `ok` → `shouldEscalate`; **pass** → `record` the served cheap `request_log` (`escalated=false`, `quality_signal=score`) + return `cheap.wire`; **escalate** → `runBufferedChain(escalation, signal)` → `record` the **served** escalation `request_log` via `escalationMeta[servedIndex]` (its tierKey/price, `escalated=true`, `quality_signal=score`), then `recordAttempt(requestId, cheapCtx, cheapOutcome)` for the superseded cheap call, + return; total escalation failure → one `error` `request_log` + throw. If cheap `!ok`: **client disconnected** (`signal.aborted`) → stop; else escalate with `quality_signal=0` (no cheap ledger row — no billable usage).
- [x] 3.3 `stream` (streaming, cascade): run `runBufferedChain(cheap, cheapSignal)` (buffered — nothing forwarded). **pass** → `replayBufferedStream(client, cheap.response, opts)`; on `{failed:true}` → **escalate** (safe — nothing sent); else `record` the cheap `request_log` on `outcome.then` — **usage/outputChars from `cheap.response`** (the billed call), delivery **status** from the outcome — and return `frames`. **escalate** → `openStreamChain(escalation, opts)`; on the served `outcome.then` `record` the served `request_log` (`escalated=true`) then `recordAttempt(requestId, cheapCtx, {status:'success', providerUsage: cheap.response.usage, …})` for the superseded cheap call; return frames (or throw the pre-commit error). **client disconnected during cheap** → stop. Only one tier ever reaches the client (invariant 3).
- [x] 3.4 Recording plumbing: add `escalated?: boolean` + `qualitySignal?: number | null` to `RecordOutcome`; thread `RequestRecorder` → `RequestLogDraft` → `toRow` (insert `escalated ?? false`, `qualitySignal ?? null`). `RequestRecorder.record(...)` **returns the allocated request id**; add `recordAttempt(requestLogId, ctx, outcome): void` that enqueues a `RequestAttemptDraft` (price resolved in the writer as for `request_log`). Cascade `routing_reason` names cheap-served / escalated→served-tier + score. Non-cascade callers write no attempt rows — #10–#13 `request_log` recording unchanged.
- [x] 3.5 **Schema + migration** (`packages/shared/src/server/db/schema.ts`): add `request_attempt` — `id`, `requestLogId` (FK → `request_log.id`, `onDelete:'cascade'`), `ownerUserId`/`orgId` (owned), `attemptIndex`, `tierKey`, `providerId`, `modelId`, the same token/price-snapshot/`usageEstimated`/`cost`/`status` columns as `request_log`, `createdAt`; indexes on `requestLogId` + `ownerUserId` + a `tokens_nonneg` check. `export type RequestAttemptRow`. Run `npm run db:generate -w packages/control-plane` + prettier the meta JSON.
- [x] 3.6 Persistence: add a `requestAttempts` accessor to the port + `persistence.ts` — `insertMany(principal, rows)` (owner forced from principal) and `listForRequest(principal, requestLogId)` (owned), mirroring `requestLogs`.
- [x] 3.7 `LogWriter`: add a `RequestAttemptDraft` type + a batched `request_attempt` insert path (owner-grouped, `ON CONFLICT (id) DO NOTHING`, the same bounded price-resolve as `request_log`); flushed with the same lifecycle. `RequestRecorder.recordAttempt` enqueues it.
