## 3. Control-plane: config, baseline store, router

- [x] 3.1 Add `packages/control-plane/src/proxy/routing.config.ts`: `registerConfig('routing', ‚Ä¶)` with `ROUTING_AUTO_LAYERS` (`z.string().default('structural')`), `ROUTING_STRUCTURAL_HIGH_THRESHOLD` (default 0.6), `ROUTING_STRUCTURAL_LOW_THRESHOLD` (default 0.25), `ROUTING_STRUCTURAL_BASELINE_ALPHA` (`gt(0).max(1)`, default 0.2), and `ROUTING_STRUCTURAL_WEIGHTS` (optional JSON string). - [x] 3.2 Add `packages/control-plane/src/proxy/structural/structural-baseline.store.ts`: `StructuralBaselineStore` = bounded in-process LRU (`MAX_BASELINE_ENTRIES`) + a **dedicated Redis connection** `REDIS_CLIENT.duplicate({ enableOfflineQueue:false, maxRetriesPerRequest:1 })` (disconnected ‚áí commands reject immediately, never buffer). Derive the fingerprint key **once at boot**: `fpKey = HMAC-SHA256(API_KEY_HMAC_SECRET, "polyrouter.structural.fingerprint.v1")`; per request `fpDigest = HMAC-SHA256(fpKey, canonicalSystem)` truncated to 128 bits. Baselines stored in a **per-`(userId,agentId|'-')` Redis HASH**, **field** = `fpDigest`. `read(userId, agentId, canonicalSystem): StructuralBaseline | null` ‚Äî **synchronous local LRU read, never awaits**; on a miss return `null` + enqueue one **coalesced** cold-seed (`HGET`). `observe(userId, agentId, canonicalSystem, x, alpha): void` ‚Äî update local EWMA synchronously + enqueue a **throttled, coalesced** Redis sync (‚â§ one write per key per `OBSERVE_FLUSH_MS`). The Redis sync is an atomic Lua: EWMA the field, **refuse a new field when `HLEN ‚â• MAX_FINGERPRINTS_PER_AGENT`**, `EXPIRE` the hash with `BASELINE_TTL`. The worker bounds **all transient state together** under `MAX_BACKGROUND_ENTRIES` and **rejects admission before allocating per-key state** when full; all errors swallowed; never throws. Implement `OnApplicationShutdown` ‚Äî cancel timers, clear queued state, `disconnect()` the duplicate connection.
